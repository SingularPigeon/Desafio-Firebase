# Desafío-firebase

Realización de un CRUD para el crear, editar, eliminar y listar usuarios. Utilizando Firestore, Pinia y Vue Router.
![](/public/preview-usuarios.png)

## Dependencias

- Vue 3
- Vue Router
- Pinia
- Firebase
